・なぜ2乗しているのか

→残差をそのまま足し合わせると正負が打ち消し合ってしまうため２乗します。

・なぜ最後に,2で割っているのか

→二乗和の式を微分すると「２」が出てくるので、それを２で割ることにより消しています。

例えば

```math
y = (x - a_1)^2 + (x - a_2)^2
= 2x -2(a_1 + a_2)x + a_1^2 + a_2^2
\frac{dy}{dx}} = 4x -2(a_1 + a_2)
2x = (a_1 + a_2)
 x = (a_1 + a_2)/2
```
